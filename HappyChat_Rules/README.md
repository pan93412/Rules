# HappyChat 規定
## 原則
- 純聊天
- 可以討論程式、也可以聊自己想要聊的話題。
## 規則
### I. 用戶規則
以不侵犯到他人權益為基礎。
- **不能** kick 任何人
- **不能**辱罵
- 在加入任何用戶前，**請先經過管理員同意。**
### II. 話題規則
- **不能** __NSFW__
- **不能**散布盜版資訊。
- **不能**討論違反中華民國法律的內容。
### III. 管理員規則
- **不能**無理由踢人。
  - **若情況危急時可踢人**，但若踢錯用戶請自行辭職。
- **不能**濫用管理員權限。
#### III#. 變更規定原則
- 當規定有任何**重大**錯誤時
- [for **一般用戶**] 若有任何規定問題，**請發 issues 以方便集中管理問題**。
### IV. 記事本規則
- 參閱 _I_ 與 _II_ 規則。
- 不能惡意在他人記事本下留垃圾訊息。
